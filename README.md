# 📄 Desafío Mi CV en HTML

Proyecto desarrollado como parte del desafío **"Mi CV en HTML"**, enfocado en la creación de una página web básica utilizando únicamente **HTML semántico**.

El objetivo principal fue construir un **Curriculum Vitae digital** junto a una página adicional de **Portafolio**, aplicando estructura web, enlaces internos, imágenes, listas y tablas.

## ⚙️ Cómo Ejecutarlo

1. Clona este repositorio:

```bash
git clone https://github.com/jleival/esafio1-mi-CV-en-html.git
```

2. Ingresa a la carpeta:

```bash
cd nombre-del-repo
```

3. Abre el archivo:

```bash
index.html
```

---

## 🌐 Publicación en GitHub Pages

1. Ir a **Settings**
2. Seleccionar **Pages**
3. Branch: `main`
4. Folder: `/root`
5. Guardar
